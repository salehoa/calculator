## دليل الاستخدام

### استخدام الحاسبة
1. أدخل التعبير الرياضي باستخدام الأزرار أو لوحة المفاتيح
2. استخدم الدوال العلمية من الصف العلوي
3. اضغط "=" أو Enter للحصول على النتيجة
4. يمكنك النقر على أي عملية في السجل لتحميل نتيجتها

### رسم الدوال
1. انتقل إلى تبويب "الرسوم البيانية"
2. أدخل الدالة بدلالة x (مثال: `sin(x)` أو `x^2`)
3. حدد نطاق قيم x (من - إلى)
4. اضغط "رسم الدالة"
5. مرر الماوس على الخط لرؤية القيم

### أمثلة على الدوال
- دوال مثلثية: `sin(x)`، `cos(x)`، `tan(x)`
- دوال كثيرة الحدود: `x^2`، `x^3`، `x^2 + 2*x + 1`
- دوال أسية: `e^x`، `2^x`
- دوال لوغاريتمية: `log(x)`، `ln(x)`
- دوال جذرية: `sqrt(x)`، `sqrt(x^2 + 1)`
- دوال كسرية: `1/x`، `1/(x^2 + 1)`
- دوال مركبة: `sin(x) + cos(x)`، `x * sin(x)`
